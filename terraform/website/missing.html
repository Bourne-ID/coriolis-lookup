Coriolis.io URL Expander

A quick, temporary URL expander for Coriolis created by BourneID using AWS serverless and functionless applications.

Coriolis is a well used Elite ship build planner, with the short code used extensively across the web. This is a proof of concept for a new URL shortener and expander using AWS services to minimise costs and maximise performance.

How to use

Navigate using the path for example: https://dev.bourne.id/1234
